h1. Moebius

Script para convertir archivos de Comic Book (.cbr y .cbz) en PDF para leerlos en varios.

h2. Uso

<pre>
<code>
	cd moebius/
	./moebius.rb example/Y-The_Last_Man-001-Excerpt.cbz
</code>
</pre>


El archivo _Y-The_Last_Man-001-Excerpt.pdf_ salire en la carpeta de moebius.

h2. Requisitos

* Ruby 1.8
* Gems: rubygems, prawn
* Unzip (para .cbz)
* Unrar (para .cbr)
