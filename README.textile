h1. Moebius

Script to convert Comic Book files (.cbr and .cbz) in PDF for portable reading.

h2. Usage

<pre>
<code>
	cd convert_comics_installation

	./moebius.rb comic_file.cbr
</code>
</pre>

It'll output a _comic_file.pdf_ file in your current directory.

h2. Requirements

* Ruby 1.8
* Unzip
* Unrar
